# ENGLISH

## Parking Meter

A simple parking meter simulator that calculates the remaining time and the change based on the amount entered by the user.

This project is part of the EBAC course **"Full Stack JavaScript"**, and its main goal is to practice the use of **classes, methods, and attributes** in JavaScript.

### How it works

- The user enters a value in dollars;
- The application calculates the remaining parking time and the change;
- The time is determined according to the price table displayed on the page.

### Technologies

- HTML
- CSS
- JavaScript

# Português

## Parquímetro

Um simulador simples de parquímetro que calcula o tempo restante e o troco com base no valor informado pelo usuário.

Este projeto faz parte do curso **"JavaScript Full Stack" da EBAC** e tem como objetivo praticar o uso de **classes, métodos e atributos** em JavaScript.

### Como funciona

- O usuário informa um valor em dólar;
- O sistema calcula o tempo restante e o troco;
- O tempo é definido de acordo com a tabela exibida na página.

### Tecnologias

- HTML
- CSS
- JavaScript

### Demo

Open the `index.html` file in your browser to run the project.
